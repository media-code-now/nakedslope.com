# NakedSlope.com

**No Fluff. Just Gear.**

Honest, independent gear reviews for skiers, snowboarders, surfers, and overlanders.

## 🎯 What We Do

We test outdoor gear and tell you what actually works. No sponsored content, no affiliate bias, no bullshit.

### Categories

- **Ski & Snowboard** — Helmets, boots, skis, goggles, bindings
- **Surfing** — Wetsuits, surfboards, fins, leashes
- **Overlanding** — Recovery gear, rooftop tents, fridges, build guides

### Free Tools

- **Ski Length Calculator** — Height + weight + terrain → exact ski size
- **Wetsuit Thickness Calculator** — Water temp + sensitivity → right mm
- **Live Conditions** — Real-time surf swell and resort snowfall (updated every 15 min)

## 🏔️ Editorial Standards

1. **No affiliate bias** — We link to the best product for you, not the highest commission
2. **Research-backed** — Every pick is built from specs, lab tests, and field reports
3. **We call out garbage** — If a $300 product isn't worth $150, we say so
4. **Updated yearly** — Gear evolves. We re-evaluate every season
